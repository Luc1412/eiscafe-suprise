# Eis-Café Surprise

Modern single-page website for Eis-Café Surprise in Berlin-Weißensee, built with Next.js, Tailwind CSS v4, and Bun.

## Stack

- Next.js 16
- React 19
- Tailwind CSS v4
- Bun
- ESLint + Prettier

## Getting Started

```bash
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Reference

Existing website: [eiscafe-surprise.de](https://www.eiscafe-surprise.de/)

## Scripts

```bash
bun run dev
bun run build
bun run typecheck
bun run lint
bun run format:check
```
